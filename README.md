# GIT-JS

GIT-JS is a beginner-level JavaScript project that demonstrates basic GIS-related calculations using Node.js.

## Features
- Calculate distance between two geographic coordinates
- Simple JavaScript implementation of GIS concepts
- Beginner-friendly code

## Technologies Used
- JavaScript
- Node.js
- Git

## Example
The project calculates the distance between two locations using latitude and longitude coordinates.

Example:
Hyderabad → Delhi distance calculation.

## Purpose
This project is created for learning Git, JavaScript, and basic GIS concepts.
